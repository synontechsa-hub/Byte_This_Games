# 🕹️ Byte This Games

**CODE THE FUTURE. BREAK THE LIMITS.**

Byte This Games is an independent game studio crafting immersive simulations, experimental RPGs, and AI-driven experiences at the bleeding edge of vision and code.

## 🌌 The Studio
Byte This Games operates as the dedicated gaming arm of the SoulLink ecosystem. We build worlds that breathe and systems that challenge, with an obsessive focus on technical precision and narrative autonomy.

## 🚀 Key Projects
- **[Linker.AI](https://bytethisgames.github.io/linkerai.html)**: A cyberpunk spatial simulation featuring AI-powered NPCs that remember, react, and evolve.
- **[Malevolent: Scorned](https://bytethisgames.github.io/synquest.html)**: A grimdark text-based RPG where every choice leaves a scar on a persistent world.

## 🛠️ Stack
- **Frontend**: HTML5, CSS3 (Custom HUD-driven UI), JavaScript (ES6+).
- **Backend**: Python (Logic engines), PostgreSQL/SQLite (State management).
- **AI**: Custom LLM integration for dynamic storytelling and NPC behavior.
